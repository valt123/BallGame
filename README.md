# BallGame
Game where you catch falling balls

You need to keep the catcher within the green area.

You have 100 hp
Green ball:
  Heals you if you let it through
  Heals: 10 hp
  Spawn chance: 5%
  
Red ball:
  Damage: 20
  Spawn chance: 84%
  
Brown ball
  Damage: 50
  Spawn chance: 10%
  
Black ball
  Damage 100
  Spawn chance: 1%
